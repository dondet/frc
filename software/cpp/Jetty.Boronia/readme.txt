====================================================
JETTSON readme.txt
====================================================


============================
Overview:  The JETTSON is a NAVIDIA image processing unit; model: TK1.  
It runs open CV along with a bunch of other libraries to accomplish a live image processing pipeline that runs independently on the robot. 
===========================


============================
Files:  “9-ic-pipeline” and “8-ic-pipeline” are the pipelines for tracking different reflective objects. 

	++++++
	“9-ic-pipeline”: is for tracking a green led circle. 

	++++++
	“8-ic-pipeline”:  is for tracking a peg position from 2017’s competition 
============================
